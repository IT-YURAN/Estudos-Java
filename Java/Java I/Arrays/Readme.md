### Arrays
Arrays em Java são
estruturas de dados indexadas de tamanho fixo que armazenam múltiplos valores do mesmo tipo (primitivos ou objetos)
de forma contígua na memória. <br>

Essa e a estrutura de um Array -> `Tipo[] nome e inicializam-se com new Tipo[tamanho]`

#### Exemplo de um Array que armazena Strings:

    String[] nomes =new String[5];

### Array de objectos

Para entendermos como funcionam os Arrays de objectos temos que saber oque sao objectos e como funcionam.<br>

#### Objectos
os Objectos em java sao a representacao das entidades do mundo real. Elas sao inicializadas com a palavra chave `new` ,<br>
e sao armazenadas na memoria **heap**.

O Array de objectos nao aramzena os objectos em si, mas sim a referencia do objecto na memoria.
