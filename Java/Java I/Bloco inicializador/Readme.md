# Bloco inicializador

Bloco inicializador é usado para declarar ou inicializar a parte comum de vários construtores de uma classe. Ele é executado toda a vez que o objecto é criado. <br>
O bloco inicializador contém o código que é sempre executado sempre que uma instância é criada e é executado cada vez quando um objeto da classe é criado. Existem 3 áreas onde podemos usar os blocos inicializadores: <br>

* Construtores
* Métodos
* blocos

**Observação**: O conteúdo do bloco inicializador é executado sempre que qualquer construtor é invocado (antes do conteúdo do construtor)
A ordem dos construtores de inicialização e bloco de inicializador não importa, o bloco inicializador é sempre executado antes do construtor.
