# Heranca

A heranca é o processo onde uma classe adquire os atributos e metodos de outra classe.<br>
Para que uma classe herde de outra classe usamos a palavra-chave `Extends` mais o nome da classe que gostariamos que a nossa classe herdasse! <br>
As classes que herdam são denominadas **classes filhas** e a classe principal é denominada **classe pai** <br>

### Beneficio e vantagem

1. O benefício é que não temos que escrever os metodos e variaveis duas vezes ou mais...
2. A vantagem é que permite que as **classe Filhas** tenham os seus metodos e as suas proprias variaveis
