# Interfaces
Uma ‘interface’ é um modelo que pode ser aplicado numa classe. <br>
Semelhante à herança, porem especifica o que uma classe tem ou deve fazer.<br>
Classes podem aplicar mais de uma ‘Interface’, herança está limitado a uma superclasse.<br>
As ‘interfaces’ podem conter metodos sem corpo, entao quando uma classe implementa essa ‘interface’, ela deve definir o que esse método vai fazer.
