# Method Overloading and Overriding

## Overriding/ subscrita
A subscrita de métodos é feito declarando um método numa subclasse, que já esta presente na classe pai, assim feito a classe filha pode ter a sua propria implementação. <br>

## Overloading
A sobrecarga de métodos acontece quando métodos compartilham o mesmo nome, mas tem parâmetros diferentes. <br>
