# Array VS ArrayList

Em Java, um Array é uma estrutura de dados de tamanho fixo e homogêne que armazena elementos do mesmo tipo, enquanto que,
o ArrayList é um tamanho dinâmico, parte do Java Collections Framework e é usado para armazenar objetos com métodos integrados para manipulação. A principal diferença entre array e ArrayList é:

* **Array**: Arrays têm um tamanho fixo e proporcionam um desempenho rápido.
* **ArrayList**: ArrayList é dinâmico em tamanho e fornece mais flexibilidade e métodos embutidos.

## Diferença entre array e ArrayList

Um Array é uma funcionalidade básica fornecida pelo Java. O ArrayList faz parte do framework de coleção em Java. Portanto, os membros do array são acessados usando [], enquanto o ArrayList tem um conjunto de métodos para acessar elementos e modificá-los. <br>
O Array é uma estrutura de dados de tamanho fixo, enquanto o ArrayList não. Não é preciso mencionar o tamanho da ArrayList enquanto cria seu objeto. Mesmo se especificarmos alguma capacidade inicial, podemos adicionar mais elementos. <br>
Uma matriz pode conter tanto tipos de dados primitivos quanto objetos de uma classe, dependendo da definição da matriz. No entanto, o ArrayList suporta apenas entradas de objetos, não tipos de dados primitivos. <br>
Como o ArrayList não pode ser criado para tipos de dados primitivos, os membros do ArrayList são sempre referências a objetos em diferentes locais de memória. <br>
