# Set & HashSet

Set é uma interface e não uma classe normal, então, não podemos criar instancias de interfaces, e só podemos criar instancia de classes que implementam essas interfaces <br>
**Não podemos acessar os elementos por Index**
## Classes que implementam a interface Set
* HashSet: Um conjunto que armazena elementos exclusivos sem qualquer ordem específica, usando uma tabela de hash e permite um elemento nulo.
* EnumSet: Um conjunto de alto desempenho projetado especificamente para tipos de enum, onde todos os elementos devem pertencer ao mesmo enum.
* LinkedHashSet: Um conjunto que mantém a ordem de inserção enquanto armazena elementos exclusivos.
* TreeSet: Um conjunto que armazena elementos exclusivos em ordem classificada, seja por pedido natural ou um comparador especificado.

## Quando usar?
Usamos que quisermos uma coleção de intens e não quisermos duplicantes
