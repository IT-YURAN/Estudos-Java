# Expressões lambda

Expressões de lambda Java, introduzidas no Java 8, permitem que os desenvolvedores escrevam código conciso e de estilo funcional representando funções anônimas. Eles permitem passar código como parâmetros ou atribuí-lo 
a variáveis, resultando em programas mais limpos e legíveis. <br>

**As expressões do Lambda permitem tratar a funcionalidade como argumento de método ou código como dados.**

Precisa usar uma interface funcional ou usar uma interface funcional pre-definida, são basicamente interfaces que contem somente um método abstrato <br>

Formula de lambda:
`(argumentos) -> {instruções}`
